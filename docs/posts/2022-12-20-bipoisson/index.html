O c√≥digo abaixo ir√° calcular os scores (rating) do times utilizando o pacote EloRating. A f√≥rmula de atualiza√ß√£o dos scores depende um fator atualiza√ß√£o \(k\).   mutate(
    winner = case_when(
      team1_score >= team2_score ~ team1,
      TRUE ~ team2
    ),
    loser = case_when(
      team1_score < team2_score ~ team1,
      TRUE ~ team2
    ),

  winner = data$winner,
  loser = data$loser,
  Date = data$date,
  draw = data$draw,
  k = 20
)

  mutate(

      eloobject = elo_fit,

      eloobject = elo_fit,

Neste segundo bloco, s√£o computadas as probabilidades de vit√≥ria do time 1, empate e vit√≥ria do time 2. Para isso, s√£o utilizadas as predi√ß√µes anteriores como o par√¢metro (\(\lambda\)) da distribui√ß√£o de Poisson e s√£o calculadas as probabilidades de cada time marcar 0 a 10 gols. Multiplicando as probabilidades s√£o obtidas as probabilidades de cada placar poss√≠vel. Por √∫ltimo, a probabilidade do time 1 ganhar √© calculada como a soma das probabilidades dos placares nos quais o time 1 marca mais gols que o time2. As probabilidades de empate e vit√≥ria do time 2 s√£o calculadas de forma similar.

preds <- map_df(seq_len(nrow(test)), ~ {
  lambda1 <- pred1[.x]
  lambda2 <- pred2[.x]

  prob1 <- dpois(0:10, lambda1)
  prob2 <- dpois(0:10, lambda2)
  probs <- outer(prob1, prob2)
  probs <- probs / sum(probs)

  prob_team1 <- sum(lower.tri(probs) * probs)
  prob_team2 <- sum(upper.tri(probs) * probs)
  prob_draw <- sum(diag(probs))

  data.frame(
    .pred_team1 = prob_team1,
    .pred_draw = prob_draw,
    .pred_team2 = prob_team2
  )
})
